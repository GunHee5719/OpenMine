## OpenMine
# Project Name
 - Open Mine : Automatic Minesweeper Gameplay Program

# Project Goal
 - To make program that automatically resolves Minesweeper that one of the basic games provided in Window 7, by recognizing monitor screen as OpenCV

# Development Tools
 - OpenCV 3.4.3
 - Java JDK 11.0.1
 - IntelliJ IDE
